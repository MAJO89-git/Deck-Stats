# Deck Stats

Ett C# konsolprogram för att registrera matcher och analysera statistik för Magic: The Gathering.

## Bakgrund

Detta var ett av mina första egna programmeringsprojekt och fungerade som en introduktion till filhantering och databehandling i C#.

Programmet läser matchdata från `.txt` filer och sammanställer statistik.

## Teknologier

- C#
- .NET
- Filhantering
- LINQ

## Funktioner

- Registrera matcher
- Läsa matchdata från textfiler
- Visa statistik
